# CyberGuard: Phishing Link Detector

## Overview
This project is part of the Lenovo Leap Cybersecurity Program. It is a Python-based utility designed to identify potential phishing URLs by analyzing common red flags in domain structures.

## Problem Statement
Users often fail to identify phishing websites, leading to data theft and scams. This tool addresses the need for automated link verification.

## Features
- Detects long URLs and suspicious subdomains.
- Identifies IP-based URLs and sensitive keywords.
- Provides immediate risk assessment.

## Technology Stack
- Language: Python 3.x
- Libraries: re (Regular Expressions)

## How to Run
1. Install Python on your system.
2. Download `app.py`.
3. Run the command: `python app.py`
4. Enter the URL you want to test.
